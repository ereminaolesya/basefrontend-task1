## Live Demo

https://ereminaolesya.github.io/basefrontend-task1/

## Описание

Реализация HTML-документа в виде цифровой карточки портфолио веб-разработчика с использованием исключительно inline-стилей для всей стилизации элементов.

Задача взята из учебного практического курса.

## Требования к задаче

Ограничения по стилизации
Только inline-стили: Запрещено использование CSS-файлов, тега <style> и CSS-классов
Единый HTML-файл: Все решение должно быть оформлено в одном файле
Фиксированная ширина: Тестирование только на экране 860px (адаптивность не требуется)

Структура портфолио
Карточка профиля с фотографией разработчика Alex Morgan
Заголовочная секция с личной информацией и цитатой
Раздел “About Me” с описанием и временной меткой обновления
